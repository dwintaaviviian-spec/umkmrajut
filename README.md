# UMKM Rajut

Website sederhana untuk menampilkan produk UMKM Rajut, berisi informasi profil usaha, katalog produk, galeri, dan kontak.

## Fitur
- Halaman Beranda
- Halaman Tentang UMKM
- Daftar Produk Rajut
- Galeri Foto
- Halaman Kontak

## Struktur Folder
- `index.html` : halaman utama
- `tentang.html` : informasi UMKM
- `produk.html` : daftar produk
- `galeri.html` : galeri foto
- `kontak.html` : informasi kontak
- `css/` : file styling
- `js/` : file JavaScript
- `images/` : gambar produk dan aset
- `videos/` : video pendukung
- `audios/` : audio pendukung

## Teknologi
- HTML
- CSS
- JavaScript
